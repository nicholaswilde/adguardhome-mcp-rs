# DHCP Management

## Objective
Enable management of AdGuard Home's internal DHCP server.

## Scope
- List current DHCP leases.
- Add/Remove static leases.

## Tools to be added
- `list_dhcp_leases`: Get dynamic and static leases.
- `add_static_lease`: Map a MAC address to a specific IP.
- `remove_static_lease`: Delete a static assignment.
