# Custom Rules Management

## Objective
Enable AI models to manage custom DNS filtering rules in AdGuard Home.

## Scope
- Retrieve existing custom rules.
- Set/Update custom rules list.
- Add/Remove specific rules.

## Tools to be added
- `list_custom_rules`: Returns the full list of custom filtering rules.
- `set_custom_rules`: Replaces the current list of rules.
- `add_custom_rule`: Appends a rule to the list.
- `remove_custom_rule`: Removes a rule from the list.
